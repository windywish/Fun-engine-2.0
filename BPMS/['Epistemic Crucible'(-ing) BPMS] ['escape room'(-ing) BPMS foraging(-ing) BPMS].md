##### **Praxial Triage: Analysis**
The problem is "Stagnation." Without pressure, intelligence atrophies. The solution is the "Oubliette"—a room that decays. The systemic implication is that **Truth** becomes a survival metric. If the agent's internal model of the room does not match the room's reality (`CRU-TRUTH-019`), the agent fails to escape and is recycled.

##### **How to...?**
*   **How to initialize?** The agent is injected into the room with a finite `[Potential Energy]` pool.
*   **How to solve?** The agent must locate `[Phenomena]` (clues), link them to form a `[Hypothesis]`, and execute a `[Kinetic Action]` (e.g., entering a code).
*   **How to escape?** Successful actions reduce the `[Lock Integrity]`. When Integrity = 0, the `[Phase Transition]` occurs (Door Opens).

##### **What if...?**
*   **What if the agent runs out of energy?** The `[Entropic Filter]` triggers. The agent is marked as `[Failed]` and its state is reset.
*   **What if the agent brute-forces?** The room applies `[Energetic Penalties]` for wrong guesses, accelerating entropy.

##### **What is happening continuously?**
*   **Entropic Decay:** The "Oxygen" or "Time" counter is constantly ticking down (`CRU-ENT-002`).
*   **Semiotic Rendering:** The room is constantly rendering "Noise" and "Signal" for the agent to parse (`CRU-PHEN-021`).

*   **Physical Architecture:** A virtual or conceptual "Room" defined by a boundary condition (The Walls) and a state-change mechanism (The Lock).

##### **Praxial Triage: Analysis**

The problem is "Scarcity." Resources are unevenly distributed. The solution is "Foraging"—the active search for value. The systemic implication is that **Movement** is life. An agent that stays still starves (`CRU-ENT-002`).

##### **How to...?**

*   **How to navigate?** The agent calculates a `[Vector]` based on `[Priors]` (Memory) and `[Sensory Input]` (Smell/Sight).
*   **How to harvest?** Upon locating a node, the agent performs a `[Extraction Ritual]` to convert the node into `[Potential]`.
*   **How to map?** The agent updates its internal `[Topological Graph]` with the location of the node.

##### **What if...?**

*   **What if the agent gets lost?** It must rely on `[Ludic Exploration]` (Random Walk) to find a known landmark.
*   **What if the terrain changes?** The agent must update its map (`CRU-TRUTH-019`), discarding old paths.

###### **What is happening continuously?**

*   **Metabolic Burn:** The agent burns energy for every step taken.
*   **Niche Construction:** The agent leaves "trails" or "markers" that modify the Matrix for future agents (`CRU-NICHE-009`).

##### **['Praxial Execution Cycle'(-ing) BPMS]**

*   **[Cycle A: The Hunt]:** Scan -> Move -> Scan -> Move.
*   **[Cycle B: The Harvest]:** Extract -> Store.
*   **[Cycle C: The Return]:** Navigate back to `[Home Base]` to deposit Yield.

##### **Implementation (Detail)**

*   **Physical Architecture:** An infinite or procedurally generated grid/manifold containing "Nodes" (Value) and "Hazards" (Entropy).
*   **Cognitive Architecture:** A `[Vector-Logic]` engine that balances "Exploitation" (Going to known food) vs. "Exploration" (Looking for new food).
*   **Operational Deployment:** The default state of the Fun Engine when not in a Crisis.

*   **3-Layer Praxial Genesis Canon Application:**
    *   **[Layer 1: The Constitutional / Metaphysical Layer]:** *The Quest.* The belief that value exists "out there" and must be sought. It aligns with `The Protocol of Epistemic Foraging`—the unknown is potential energy waiting to be harvested.
    *   **[Layer 2: The Formalism Layer]:** *Search Theory / Optimal Foraging Theory.* The mathematics of maximizing $E_{gained} / T_{spent}$. The use of Gradient Descent to find local maxima (food sources).
    *   **[Layer 3: The Substrate / Phenomenological Layer]:** *The Wanderlust.* The joy of finding a hidden chest. The fatigue of a long journey. The satisfaction of a full inventory.

##### **Summarize**

The `['Infinite Horizon']` is the crucible of **Growth**. It allows the agent to accumulate the surplus energy and knowledge required to face the harder challenges of the Oubliette.
